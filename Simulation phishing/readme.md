# Simulateur de Phishing

Ce script illustre, de manière totalement sécurisée et pédagogique, **les mécanismes fondamentaux du phishing**.
Il utilise une interface graphique Tkinter pour présenter trois éléments essentiels :

1. un faux e-mail typique de phishing,
2. une fausse page de connexion factice,
3. une liste de bonnes pratiques pour éviter les attaques réelles.

Aucun renseignement n’est collecté : tout est simulé localement pour aider à **comprendre, reconnaître et éviter** les tentatives de phishing.

## Objectifs du script

* Montrer à quoi ressemble un e-mail de phishing classique.
* Simuler une fausse page de connexion pour reconnaître les signes d’arnaque.
* Apprendre les bonnes pratiques pour se protéger.
* Fournir un outil simple de **sensibilisation**, adapté à un cadre éducatif.

## Fonctionnalités principales

### 1. Simulation d’e-mail de phishing

Affiche une fenêtre Tkinter contenant un message copiant les formulations typiques :

* urgence artificielle,
* menaces sur le compte,
* faux lien incitant à cliquer.

Cela permet d’apprendre à repérer les schémas récurrents de manipulation.

### 2. Simulation d’une fausse page de connexion

Affiche une imitation simplifiée d’un formulaire frauduleux.

L’objectif : comprendre qu’une page peut sembler légitime alors qu’elle ne l’est pas.

### 3. Conseils de prévention

Présente une liste claire de bonnes pratiques, incluant :

* vérification de l’expéditeur,
* prudence avec les liens,
* repérage des fautes,
* utilisation de l’authentification à deux facteurs,
* mise à jour des protections.

## Interface graphique

Le script ouvre une fenêtre Tkinter contenant trois boutons :

* **Simuler un faux e-mail**
* **Simuler une fausse page de connexion**
* **Afficher les conseils de prévention**

Chaque bouton déclenche une boîte de dialogue explicative.

## Intérêt en cybersécurité

Ce simulateur permet de comprendre :

* les mécanismes psychologiques exploités par les attaquants,
* l’importance d’analyser un message avant d’agir,
* les signes les plus fréquents d’un phishing,
* les bonnes pratiques de défense.
